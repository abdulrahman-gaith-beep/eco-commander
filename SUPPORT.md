# Getting help

## Supported environment

- macOS on Apple Silicon or Intel.
- Bash 5+ recommended (core CLI works on macOS default bash 3.2), Git, Homebrew, SwiftBar for menu-bar usage, and Python 3.10 through
  3.13 for poller and scheduler modules.
- The `main` branch and the latest tagged release are the supported code lines.

## Support channels

- **Documentation:** start with [`docs/INDEX.md`](./docs/INDEX.md).
- **Troubleshooting:** [`docs/getting-started/troubleshooting.md`](./docs/getting-started/troubleshooting.md).
- **Bugs / feature requests:** open a GitHub issue using the templates in
  `.github/ISSUE_TEMPLATE/`.
- **Security issues:** see [`SECURITY.md`](./SECURITY.md).
- **Direct contact:** 245627891+abdulrahman-gaith-beep@users.noreply.github.com.

## Response expectations

- Security reports: acknowledgement within 72 hours.
- Reproducible bugs: best-effort triage when a failing command, expected output,
  actual output, and commit SHA are included.
- Feature requests: reviewed when they describe the operator workflow and the
  tradeoff they introduce.

## Unsupported requests

- Debugging private credentials, private logs, or personal machine data in a
  public issue.
- Support for Python 3.14 until project dependencies are verified there.
- Changes that require weakening repository security or disclosure controls.
